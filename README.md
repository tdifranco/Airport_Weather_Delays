# IA626 Final Project: Weather and Flight Delays

## Summary / Initial Question
How do weather conditions affect flight delays and cancellations at major U.S. airports?

## Data Sources
- Flight on-time performance dataset
- Historical weather dataset

## Why These Sources
These sources allow comparison of operational air travel outcomes against environmental conditions.

## ETL Process
1. Download flight data
2. Pull weather data
3. Clean timestamps
4. Map airports to stations
5. Aggregate weather hourly
6. Join datasets
7. Export cleaned table

## Analysis Plan
- Delay distributions by condition
- Airport-level comparison
- Seasonal analysis
- Correlation/regression or classification
